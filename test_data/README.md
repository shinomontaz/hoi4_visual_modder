# Test Data

This directory contains test files for development and testing.

## Structure

- `focus_trees/` - Sample national focus tree files (.txt)
- `technologies/` - Sample technology files (.txt)

## Note

The `test_tech.txt` file in the project root is also used for testing.

## Adding Test Files

When adding test files:
1. Use real HOI4 mod files when possible
2. Include both simple and complex examples
3. Document any special cases or edge cases
